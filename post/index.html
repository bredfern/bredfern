creative chromeos

Back when Chrome Os first came out Steve Ballmer was still the CEO of Microsoft. He basically wrote these machines off as fancy paper weights. But when you run a chrome os device in developer mode there is actually quite a lot you can do. Before boot load locking became the norm with windows devices it was easy to buy a windows machine and then entirely blow off windows for Linux.

2016-09-24

frequency scaling

You actually need to roll your own kernel on chromeos for this but for raspberry pi3 its pretty easy to switch off the power management. For a laptop you might not want to do this, it will burn your battery faster. But if you want full performance you need to shut off the frequency scaling. Also having a low latency kernel build helps. Its easier to do this on the pi3 because its not sharing a kernel with chromeos but you can build a new chromeos kernel from source, something you definitely can’t do on windows or macosx.
